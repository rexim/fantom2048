# Fantom 2048 #

2048 game written in Fantom

## Quickstart ##

* install [Fantom](http://fantom.org/)
* `$ git clone https://github.com/rexim/fantom2048 && cd fantom2048`
* to build: `$ fan src/build.fan`
* to run:   `$ fan fantom2048`
* to test:  `$ fant fantom2048`
